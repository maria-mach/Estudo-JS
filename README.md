# Projeto Jujutsu Kaisen

Site estático inspirado em *Jujutsu Kaisen*, criado com HTML, CSS e JavaScript.

## Visão Geral

O projeto apresenta uma página interativa sobre o universo do anime/mangá. Ele inclui:
- Seção de sinopse e tópicos do universo
- Navegação de personagens com tabela de informações
- Exibição de clãs com informações clicáveis
- Seleção de personagem com vídeo e áudio
- Formulário de comentários
- Menu responsivo para dispositivos móveis

## Estrutura do Projeto

- `index.html` - página principal do site
- `estilos/style.css` - estilos e layout do projeto
- `script/infos.js` - dados dinâmicos de conteúdo (personagens, clãs, universo)
- `script/script.js` - lógica de interação da página
- `imagens/` - imagens usadas na página
- `videos_audio/` - pasta para possíveis vídeos e áudios de apresentação

## Principais Funcionalidades

- Conteúdo interativo nos tópicos do universo
- Carrossel de personagens com navegação anterior/próximo
- Detalhes de personagens exibidos em tabela
- Área de informações sobre clãs com clique para exibir textos
- Seleção de personagem que altera vídeo e áudio
- Comentários do usuário via formulário
- Menu móvel mostrado/oculto automaticamente com largura de tela

## Como Executar

1. Abra o arquivo `index.html` no seu navegador.
2. Alternativamente, use uma extensão como **Live Server** no VS Code para visualizar localmente.

## Observações

- Este é um site estático, sem backend.
- A pasta `videos_audio/` pode ser usada para adicionar mídias extras.
- Caso não veja o vídeo ou áudio, confirme se os arquivos são referenciados corretamente no HTML ou JavaScript.

## Ferramentas Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

Projeto desenvolvido como atividade de estudo de HTML, CSS e JavaScript.
